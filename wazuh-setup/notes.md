# Wazuh Mini SOC Lab Setup

## Objective
Set up a basic SOC lab using Wazuh to monitor a Windows endpoint and collect security events.

## Tools Used
- Oracle VirtualBox
- Ubuntu
- Wazuh Server
- Wazuh Dashboard
- Wazuh Agent for Windows

## What I Completed
- Installed Ubuntu in VirtualBox
- Installed Wazuh server, dashboard, and indexer
- Configured networking between host Windows and Wazuh VM
- Installed Wazuh agent on Windows
- Verified the Windows endpoint appeared as active in Wazuh

## Skills Practiced
- Virtual machine setup
- Linux terminal basics
- SIEM deployment
- Endpoint monitoring
- Agent deployment
- Basic security lab documentation
